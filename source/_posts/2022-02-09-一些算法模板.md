# 一些算法模板

```c++
void binarysearch(int key)
{
    int l = 0, r = n - 1;
    while(l < r)
    {
        int mid = (l + r) >> 1;
        if(a[mid] >= key)
            r = mid;
        else
            l = mid + 1;
    }
    if(a[l] != key)
    {
        cout << "-1" << " ";
        return;
    }
    else
    {
        cout << l + 1 << " ";
    }
}
```
